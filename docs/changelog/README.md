# Changelog Semanal

Diario de bordo do projeto JIT Manager. Um arquivo por semana.

## Formato

```
YYYY-WNN.md   (ano-semana)
```

Cada arquivo contem:
- **O que foi feito** — entregas e avancos concretos
- **Decisoes tomadas** — referencia ADRs quando aplicavel
- **Proxima semana** — prioridades imediatas
