# Company docs — JIT Manager

Documentos-base para direcionamento empresarial da JIT Manager.

## Arquivos
- `source-of-truth.md`: base canônica de mensagem, ICP e oferta
- `direcionamento.md`: definição oficial da empresa, posicionamento e fronteiras
- `foco-principal.md`: ICP, dores, oferta principal e escopo inicial
- `criterios-cnpj.md`: critérios objetivos para abrir CNPJ com base em evidências

## Como usar
1. Comece por `source-of-truth.md`
2. Use `direcionamento.md` para visão de negócio e fronteiras
3. Use `foco-principal.md` para execução comercial e qualificação
4. Avalie `criterios-cnpj.md` ao final do ciclo de MVP
