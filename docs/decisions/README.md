# Registro de Decisoes (ADRs)

Cada decisao arquitetural ou estrategica importante vira um arquivo aqui.

## Formato

```
NNN-titulo-curto.md
```

Campos obrigatorios:
- **Data** — quando foi decidido
- **Status** — Proposta | Aceito | Substituido | Descontinuado
- **Contexto** — por que surgiu essa questao
- **Decisao** — o que foi decidido
- **Consequencia** — o que muda por causa dessa decisao
